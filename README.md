# Citadel Vigil

Desktop idle tower defense game set in a dieselpunk universe.

## 30-Day Challenge
Building this game in 30 days (Nov 22 - Dec 22, 2024) as my first commercial indie game.

## Features (planned)
- Desktop companion app (always-on-top window)
- Idle resource generation
- Auto-defense against monster waves
- Tech tree progression
- Prestige system

## Tech Stack
- Godot 4
- 2D pixel art / low-poly aesthetic
